Online Chat Room Server

How to Run:
1. Open the project
2. Open server/server.py.
3. Run server.py before running any clients.

The server listens on:
Host: 127.0.0.1
Port: 12345

The server supports:
- Multiple clients using threading.
- User login and registration.
- Public messages.
- Direct messages.
- Exit/logout.
- Active user updates.

Registered users are stored in users.json.
