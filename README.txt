Online Chat Room Project

This project implements a TCP-based online chat room using Python 3.

Project Structure:
client/
    client.py
    README.txt

server/
    server.py
    users.json
    README.txt

Features:
- TCP socket communication.
- JSON message encoding.
- Multithreaded server.
- Multithreaded client.
- Public messaging.
- Direct messaging.
- Login and registration.
- Active users list.
- Error handling.

How to Run:
1. Run server/server.py.
2. Run client/client.py in one or more terminals.
3. Log in with a username and password.
4. Use PM, DM, or EX commands.
